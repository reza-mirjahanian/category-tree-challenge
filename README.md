##  Category tree API
By Reza Mirjahanian 

### install
- MongoDB 4.4
- Node 14
- install cross-env globally
### Running
`npm i`

`npm start`

`http://localhost:3000/api/categories`

`http://localhost:3000/api/categories?id=2`

### Testing
`npm test`

`npm run coverage`
